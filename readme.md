#Como se hacer funcionar el codigo.
##Visual Code.
1. Para ejecutarlo de manera mortal se descarga la carpeta de tarea1.
2. Todos los archivos deben de estar acomodados, de tal modo que permite su ejecucion sin problemas.
3. Se puede utilizar el pluggin de live server(De no tenerlo se instala) para ver sin problemas y permite el modificado.
4. Se aparecera un logo en la parte inferior que te deja ir a verlo directamente (Go Live).
5. Se abre directamente desde el html, para que se ejecute sin problemas; de otra manera se abrira la carpeta raiz y se tendra que seleccionar el archivo index.html directamente desde ahi(Sucede si se habilida desde el archivo .js, .css o .md)
##Abrir la pagina normal(Si estas bien wey)
6. Se pulsa directamente en index.html y se abre de manera automatica.
